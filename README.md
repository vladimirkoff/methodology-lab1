# Lab1

## Installation

1. Clone this repo
```bash
git clone https://github.com/vladimirkoff/methodology-lab1.git
```
2. Run main.js file inside of the root of the project, set .md file path 
```bash
node main/main.js test/test.md
```
also there is --out <file> param.
